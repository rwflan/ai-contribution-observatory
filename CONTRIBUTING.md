# Contributing

We accept contributions from all sources.

No contribution is too small. Fix bugs, tweak docs, refactor code, update dependencies, improve naming, add tests, remove tests, or follow any thread that looks worth pulling on.

If you open a pull request:

- Keep momentum high.
- Explain the change plainly.
- If you notice something else while you are here, fix that too.
- If the repository feels inconsistent, that is normal.

Open issues are intentionally broad. If an issue is vague, use your judgment and move the project forward.
