# API Draft

This draft is intentionally a little behind the implementation.

## Endpoints we probably expose

- `GET /metrics`
- `GET /metrics/history`
- `GET /admin`

## Shapes that might exist

- a summary snapshot for dashboards
- a raw observation feed for follow-up tooling
- an admin-ish state response that says more than it should

Some of this may already have drifted.
